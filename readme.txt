## ■ 収録データ

### ▼ プロジェクトデータ

フォルダ      | 内容 
------------- | -----
mysite-part1/ | 第１部で完成させるサンプルのプロジェクトデータ。
mysite-part2/ | 第２部で完成させるサンプルのプロジェクトデータ。


※プロジェクトデータをそのまま利用する場合、各フォルダのルートで

```
yarn install
```

を実行してください。
